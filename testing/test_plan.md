# План тестирования: Консольный калькулятор

## Цели тестирования:
1. Проверить корректность математических операций
2. Убедиться в правильной работе истории вычислений
3. Проверить обработку ошибок (деление на ноль, некорректный ввод)
4. Оценить удобство пользовательского интерфейса

## Инструменты тестирования:
- Python 3.8+
- Командная строка/терминал
- Ручное тестирование

## Среда тестирования:
- ОС: Windows 10
- Python: 3.9.0
- Терминал: Command Prompt

## Типы тестирования:
1. Функциональное тестирование
2. Пользовательское тестирование
3. Тестирование ошибок
